## App UI Components

`src/lib/ui/app`

- Navbar
- SidebarNav
- BottomNavbar

In the app layout (`src/routes/(app)/+layout.svelte`), a default scafollding is provided with the SidebarNav and a Navbar + BottomNavbar when on mobile devices.

You can customize it to your needs.
