## Basic Usage

Add your own testimonials to the `Testimonials.svelte` component.

To import the component, use:
```html +page.svelte
<script lang="ts">
    import Testimonials from '$lib/ui/marketing/Testimonials.svelte';
</script>

<Testimonials />
```
