## Basic Usage

Add your own logos to the `LogoCloud.svelte` component.

To import the component, use:
```html +page.svelte
<script lang="ts">
    import LogoCloud from '$lib/ui/marketing/LogoCloud.svelte';
</script>

<LogoCloud />
```
