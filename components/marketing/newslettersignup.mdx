## Basic Usage

Add your newsletter signup logic or add your Loops form ID to the `NewsletterSignup.svelte` component.

To import the component, use:
```html +page.svelte
<script lang="ts">
    import NewsletterSignup from '$lib/ui/marketing/NewsletterSignup.svelte';
</script>

<NewsletterSignup />
```
