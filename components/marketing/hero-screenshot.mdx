## Basic Usage

Add your own copy and images to the `HeroScreenshot.svelte` component.

To import the component, use:
```html +page.svelte
<script lang="ts">
    import HeroScreenshot from '$lib/ui/marketing/HeroScreenshot.svelte';
</script>

<HeroScreenshot />
```
