## Basic Usage

Add your own copy and images to the `HeroSimple.svelte` component.

To import the component, use:
```html +page.svelte
<script lang="ts">
    import HeroSimple from '$lib/ui/marketing/HeroSimple.svelte';
</script>

<HeroSimple />
```
