---
title: "Quickstart"
description: "Start building your SaaS in under 5 minutes"
---

Follow these steps to get started with the boilerplate.

<Steps>
    <Step title="Boilerplate repo access">
        Make sure you have access to the boilerplate repo 
    </Step>
    <Step title="Clone or 'Use this template'">
    To get started clone the repo or click on the "Use this template" button.

    ```bash
    git clone https://github.com/thomasmolorg/launchleopard.git [APP_NAME]
    cd [APP_NAME]
    npm install
    ```

    </Step>
    <Step title="Set .env">
        Rename the `.env.example file` to `.env` and set the environment variables
    </Step>
    <Step title="Start the app">
        Start the app locally with the following command
        ```bash 
        npm run dev
        ```
    </Step>
</Steps>
