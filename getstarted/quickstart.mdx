Follow these steps to get started with the boilerplate.

<Steps>
    <Step title="Create a new repo with the boilerplate">
    To get started, get the repository you want to use, choose between:
   <CardGroup cols={2}>
        <Card title="Auth.js Repository" href="https://github.com/thomasmolorg/launchleopard-authjs">
            <img src="/images/tech/authjs.webp" width="36px" height="36px"/>
        </Card>
        <Card title="Supabase Repository" href="https://github.com/thomasmolorg/launchleopard-supabase">
            <img src="/images/tech/supabase.webp" width="36px" height="36px"/>
        </Card>
        </CardGroup>

    Click on `Use this template` button to create a new repository with the Launch Leopard boilerplate.
    </Step>
    <Step title="Clone your new repository">
        Clone your new repository with the following command
        ```bash
        git clone https://github.com/[your_username]/[your_repo_name].git
        ```

        Or use VSCode to clone your repository
    </Step>
    <Step title="Set .env">
        Rename the `.env.example` file to `.env`
    </Step>
    <Step title="Install dependencies">
        Install dependencies with the following command
        ```bash
        npm install
        ```
    </Step>
    <Step title="Start a local server">
        Start the app locally with the following command
        ```bash
        npm run dev
        ```
        Your app should now be running on [http://localhost:5173](http://localhost:5173), or check your terminal for the port number.
    </Step>
    <Step title="Follow tutorial">
        Now follow tutorial depending on the base repository you are using:
        <CardGroup cols={2}>
        <Card title="Auth.js Tutorial" href="/tutorials/authjs">
            <img src="/images/tech/authjs.webp" width="36px" height="36px"/>
            Follow this tutorial if you are using the Auth.js repository.
        </Card>
        <Card title="Supabase Tutorial" href="/tutorials/supabase">
            <img src="/images/tech/supabase.webp" width="36px" height="36px"/>
            Follow this tutorial if you are using the Supabase repository.
        </Card>
        </CardGroup>
    </Step>

</Steps>
