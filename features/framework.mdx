[SvelteKit](https://kit.svelte.dev/) is a web framework for building web applications of all sizes, with a beautiful development experience and flexible filesystem-based routing.

Check the [official documentation](https://kit.svelte.dev/docs) for more information.

Typescript is the default in Launch Leopard with [Zero-effort type safety](https://svelte.dev/blog/zero-config-type-safety) enabled.

Make sure you have the [Svelte for VS Code](https://marketplace.visualstudio.com/items?itemName=svelte.svelte-vscode) extension installed to get type safety.
