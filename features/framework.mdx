[SvelteKit](https://kit.svelte.dev/) is a web framework for building web applications of all sizes, with a beautiful development experience and flexible filesystem-based routing.

Check the [official documentation](https://kit.svelte.dev/docs) for more information.

Typescript is the default in Launch Leopard.
