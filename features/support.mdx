Customer support is a critical part of any business. It's how you keep your customers happy and how you learn about what's working and what's not.

The easiest way is to add a chat to your app, here are some options:
* [Atlas](https://atlas.so/) (free tier)
* [Crisp](https://crisp.chat/en/) (free tier)
* [Intercom](https://www.intercom.com/) (paid)

All of these services also allow you to create an simple dedicated help center page for tutorials and FAQs.

## Setting up a support system

1. Create account with one of the above services
2. Add the code snippet to `src/app.html` in the `<head>` tag
